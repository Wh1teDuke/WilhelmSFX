# Wilhelm SFX

![Wilhelm stares at your soul](Sheb_Wooley_1971.jpeg "Wilhelm")

## Build

Requires `dotnet 10` and `ffmpeg`. Example:

```bash
./gen.cs --sample-rate 22_050
```

- [www.nps.gov](https://www.nps.gov)
